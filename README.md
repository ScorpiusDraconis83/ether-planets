# ΞPlanets

This is the code repository for Ether Planets, a fully on-chain, procedurally generated, 3D Planets.

## Stack

### Backend

- Hardhat
- Typescript
- Typechain

### Frontend

- React
- wagmi
- RainbowKit

# ΞPlanets

## Development

### Backend

```
cd backend
yarn
yarn start
```

### Frontend

```
cd frontend
yarn
yarn start
```
