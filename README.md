# Black Holes

This is the code repository for Ether Planets, a fully on-chain, procedurally generated, 3D Planets. 

## Stack

### Backend

- Hardhat
- Typescript
- Typechain

### Frontend

- React
- wagmi
- RainbowKit

# Ether Planets

## Development

### Backend

Start node

```
cd backend
yarn
yarn start
```

### Hardhat UI

Start hardhat ui

```
yarn ui
```

### Frontend

```
cd frontend
yarn
yarn start
```
